
For $ x \notin [0,1] $  the conditional density function 
\[
    f_{Y|X} (x,y) = \frac{f_{(X,Y)} (x,y)}{f_{X} (x)}  
\] 
is not defined since the denominator $ f_{X} $ is non-positive. ($ f_{X} (x) = \int_{\R} f_{(X,Y)}
(x,y) dy = \int_{\R} 0 dy = 0$.) Thus 

\[
    \E(Y |X) = \frac{8 X^2 + 1}{6 X^2 +1} 
\] 
where $ X$ has be take values in the interval $ [0,1]$.
