
True. A cumulative distribution of a random vector is defined as 

\[
    \P(X_{1} \leq x_1, \dots, X_{n} \leq x_{n} ) 
.\] 
Thus, $ F_{(X_1, \dots, X_{n} )}(x_1,\dots,x_{n})  = (F_{X_1 }(x_{1}), \dots ,F_{X_{n} }(x_{n} )   )$
, and so as $ x_{i} $ increases, the vector $ (F_{X_{1} }(x_{1}), \dots, F_{X_{i} } (x_{i} ), \dots,
F_{X_{n} } (x_{n}  )$ increases.
