

Using lemma 5.13, we see that 

\[
    \P(X \in (a, b)) = \lim_{x \to b^{-} } F_{X} (x) - F_{X} (a)  
.\] 

Since $ a = 1.5 \in [1,2)$, we have that $ F_{X} (1.5) = 1/n$. 

Since $ b = 2.1$, as we approach $ b$ from the left in the limit $ \lim_{x \to b^{-} } $ there are
some values $ x$ that are in the interval $ x \in (2,2.1)$ and the function $ F_{X} (x) = 2/n$ for these
points so taking the limit $ x \to b^{-} $ we see that $ \lim_{x \to b^{-} } F_{X} (b) = \lim_{x \to
b^{-} } 2/n = 2/n$. 

Thus, $ P(X \in (1.5,2.1)) = 2/n - 1/n = 1/n$.
